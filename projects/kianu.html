Ресторан HTML · CSS · JS

Kianu

Современный сайт для ресторана с онлайн-меню, атмосферной галереей блюд и формой бронирования столика.

Live →
🍽️

Задача

Ресторан Kianu нуждался в современном сайте, который передаёт атмосферу заведения, показывает меню и даёт возможность забронировать столик онлайн. Задача — создать сайт, который выглядит дорого, загружается быстро и хорошо работает на мобильных.

Что реализовано

  • Главная страница с hero-блоком и атмосферной фотографией интерьера
  • Раздел онлайн-меню с категориями (закуски, основные блюда, десерты, напитки)
  • Галерея блюд с лайтбокс-просмотром
  • Форма бронирования столика с выбором даты и количества гостей
  • Блок «О ресторане» с историей и командой
  • Контакты: карта, адрес, часы работы, телефон
  • Полностью адаптивный дизайн (mobile-first)
  • Плавные CSS-анимации и переходы

Технологии

HTML5 CSS3 JavaScript CSS Grid Flexbox CSS Animations
