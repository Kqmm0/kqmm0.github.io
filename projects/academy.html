Academy

Многостраничный сайт для образовательной платформы: каталог курсов, страницы преподавателей, блог и личный кабинет студента.

Live → GitHub
📚

Задача

Academy — учебный проект многостраничного сайта для образовательной платформы. Задача — создать реалистичную архитектуру современной ed-tech платформы: с каталогом курсов, карточками преподавателей, блогом и страницей для студентов.

Что реализовано

  • Главная страница с hero, преимуществами платформы и популярными курсами
  • Каталог курсов с карточками, ценами и рейтингами
  • Страница отдельного курса с программой, описанием и CTA
  • Раздел преподавателей с профилями и списком курсов
  • Блог с карточками статей и страницей статьи
  • Страница «О платформе» с историей и командой
  • Контакты и FAQ-секция с аккордеоном
  • Адаптивная вёрстка и плавные переходы между страницами
